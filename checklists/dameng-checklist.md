# 达梦 DM8 适配检查清单

## 迁移前

- 确认 DM8 具体版本。
- 确认 JDBC 驱动版本。
- 确认 JDK 版本。
- 确认是否启用 MySQL 兼容模式。
- 确认数据库大小写策略。
- 确认业务 schema 和业务账号。
- 确认客户环境是否允许 Docker 或本地测试库。

## 开发阶段

- 不使用 MySQL 反引号。
- 不使用数据库关键字作为表名或字段名。
- MyBatis XML 不写死 MySQL 专属函数。
- MyBatis-Plus 分页显式设置 `DbType.DM`。
- 主键策略优先使用 `ASSIGN_ID`。
- 批量插入单独测试。
- CLOB/JSON 替代方案单独测试。
- 时间字段单独测试。

## 测试阶段

- 所有核心接口在 MySQL 和 DM8 各跑一遍。
- 核对分页总数。
- 核对排序结果。
- 核对时间字段。
- 核对唯一约束。
- 核对空值行为。
- 核对批量写入事务。

## 上线前

- 确认连接池配置。
- 确认最大连接数。
- 确认 SQL 日志可临时开启。
- 确认初始化脚本可重复执行。
- 确认备份和回滚方案。
- 确认客户环境达梦授权和版本。
