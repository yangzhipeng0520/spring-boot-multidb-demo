# 付费版预览：Spring Boot 多数据库适配实战包

本仓库是免费版 Demo，主要用于验证 Spring Boot 3 + MyBatis-Plus 在 MySQL、PostgreSQL、达梦 DM8 下的基础适配方式。

付费版不是另一个后台脚手架，而是一套更完整的企业数据库迁移资料包，目标是帮助 Java 后端开发者减少 MySQL 迁移达梦、PostgreSQL、国产数据库时的踩坑成本。

## 适合购买的人

- 正在做 MySQL 到达梦 DM8 迁移的 Java 开发者。
- 接到政企、信创、国产数据库适配任务的外包团队。
- 需要给团队新人准备数据库适配规范的技术负责人。
- 想快速建立多数据库兼容检查清单的后端开发者。

## 不适合购买的人

- 想要完整后台管理系统的人。
- 需要远程部署和一对一项目改造的人。
- 希望自动转换所有 SQL 的人。
- 希望覆盖所有国产数据库细节的人。
