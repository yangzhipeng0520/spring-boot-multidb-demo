- GitHub 公开仓库保留当前 Demo。
- README 中明确免费版包含 MySQL、PostgreSQL、达梦基础适配样例。

## 阶段 2：付费版内容

免费版只保留基础 CRUD 和基础 SQL。

- 20-30 个迁移差异案例。

## 阶段 4：低维护规则

- 不接远程部署。
- 不接业务定制。
- 不承诺适配客户私有 SQL。
- 每周固定一天处理 issue。
- 只修复 Demo 和文档错误。
- 重大版本每月最多发布一次。
